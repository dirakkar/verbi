# Verbi

Welcome to the source code of Verbi, the Web development platform.

Verbi is developed as a part of the [Dirakkar](https://github.com/dirakkar) project — check it out for more exciting stuff.

## Development

```sh
pnpm install # install npm dependencies
pnpm start # start the local development server
pnpm release # build and publish the packages
```
